#Projeto de API

A API que escolhi contém todos os pokemóns existentes, nessa API contém todas as habilidades dos pokemóns, imagem, altura, id, todos os ataques que ele pode aprender e seus estatos.

Para rodar a aplicação primeiro criei ela utlizando o comando "npx create-expo-app meuApp --template blank" e para conseguir rodar no meu navegador tive que utilizar o comando "npx expo install react-dom react-native-web" para instalar e para ele realmente rodar dentro do projeto utilizei o comando "npx expo start".

A funcionalidade que implementei foi basicamente de pesquisa, onde se tem um campo de pesquisa e um botão para procurar, quando ele acha um pokemón valido, ele aparece a sprite dele em baixo, junto aparece também seu peso e sua habilidade.
